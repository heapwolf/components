Intro

Component examples can be used as a starting point for your own component library or app. See the github link in each example and the docs page for more details. 

npm install @optoolco/tonic
npm install @optoolco/components
const Tonic = require('@optoolco/tonic')
const registerComponents = require('@optoolco/components')

registerComponents(Tonic) // registers all the components for you
